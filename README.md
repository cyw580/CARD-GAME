# CARD-GAME
一个卡牌游戏

## 简介

## 使用说明
开始时输入两位玩家名字

每位玩家选择自己的职业
* 使用```[↑]```/```[←]```，```[↓]```/```[→]```来移动焦点
* 使用```[Enter]```/```[SPACE]```来确定职业

在游戏过程中
* 使用```[↑]```/```[←]```，```[↓]```/```[→]```来移动焦点
* 使用```[z]```来出牌
* 使用```[x]```来弃牌，第一次按```[x]```会将图标改为```X```，再按一次进行确认
* 使用```[Enter]```/```[SPACE]```来结束自己的回合
